### 📌 Proyectos Destacados

Aquí dos de mis proyectos más representativos, donde he aplicado lógica compleja y diseño de experiencia de usuario.

#### 🦠 [**SimuladorEcosistema**](https://github.com/ThePacktur/SimuladorEcosistema) - _Django, Python_
**¿Qué hace?** Una aplicación web que simula la propagación de una enfermedad infecciosa en una población usando el modelo epidemiológico **SEIRD** (Susceptible, Expuesto, Infectado, Recuperado, Fallecido).

**¿Por qué es interesante?**
*   **Backend robusto**: Implementa los cálculos del modelo en Python con Django, demostrando capacidad para manejar lógica de negocio compleja.
*   **Visualización de datos**: Genera gráficos interactivos que muestran la evolución de la epidemia en tiempo real, facilitando la comprensión del usuario.
*   **Impacto**: Útil para entender conceptos de salud pública y el efecto de diferentes variables en una pandemia.

#### 🐱 [**DeskPets**](https://github.com/ThePacktur/DeskPets) - _Python, PyQt5_
**¿Qué hace?** Una adorable mascota virtual (¡gato, cactus o pato!) que vive en tu escritorio y te acompaña mientras trabajas. Fue mi proyecto más reciente y divertido.

**¿Por qué es interesante?**
*   **Interfaz gráfica de escritorio**: Desarrollado con **PyQt5**, demuestra mi capacidad para crear aplicaciones con interfaz gráfica más allá de la web.
*   **Programación orientada a eventos**: La mascota tiene ciclos de animación y reacciona a interacciones simples, mostrando manejo de eventos y estados.
*   **UX lúdica**: El objetivo era crear una experiencia de usuario cálida y entretenida, un pequeño respiro en la jornada laboral.
